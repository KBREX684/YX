版本：v1.1.0

## 用途

集中管理所有"需要决策但当前未定"的设计问题。替代过去散落在各模块底部的"待设计问题"节。

每条问题标注来源模块、影响范围、可选方案（如有）和当前状态。

**状态说明：**
- 🔴 阻塞：当前阶段必须决策，否则相关模块无法继续推进
- 🟠 待决：影响重要，但第一阶段内可以先用默认值推进
- 🟡 可缓：第二阶段或叙事深化阶段再决策

更新规则：决策做出后标注"已定案"并写入决策内容和关联的模块版本号更新。

## 开放问题

### Q-01 玩家具体身份来源

- **状态：** 🟠 待决
- **来源模块：** 11-narrative-worldbuilding
- **影响范围：** 叙事与世界观模块、电台 NPC 对话、第一副本进入理由
- **问题描述：** 玩家是"被旧收容机构选中的最后执行者"，但具体来源未定：候选方案包括继任者（主动接受培训）、幸存者（被迫卷入异常后被招募）、协议候选人（签署了某种协议）、被动卷入者（意外进入基地范围）。
- **可选方案：**
  - 方案 A 继任者：玩家有一定收容背景知识，有利于教程文本自然，但减少陌生感。
  - 方案 B 幸存者：玩家对基地和规则一无所知，有利于带着玩家一起探索，恐怖感更强。
  - 方案 C 协议候选人：玩家知道签了什么但不知道代价，制造叙事悬疑，但初期需要补充"协议"的世界观铺垫。
- **当前默认值：** 使用方案 B（幸存者）作为第一阶段原型叙事，电台第一通话解释"你是从异常事件中唯一存活的人，基地选中了你"。

### Q-02 电台 NPC 的真实立场与可信度

- **状态：** 🟠 待决
- **来源模块：** 10-base-management-research / 11-narrative-worldbuilding
- **影响范围：** 电台对话内容、电台关系结局变量、旧机构真相叙事结构
- **问题描述：** 电台 NPC 是"旧研究员留下的通讯人格或录音系统"，但其动机和可信度未定：是中立引导者、主动误导者、被旧机构控制的残留、还是有独立意志的人格？
- **可选方案：**
  - 方案 A 中立录音：客观播报任务，不会撒谎，但有信息盲区。玩家可信任，叙事张力来自"信息不完整"。
  - 方案 B 主动误导：电台有立场，会选择性隐瞒或扭曲信息，玩家需要交叉验证。叙事张力来自"不知道能不能信"。
  - 方案 C 双重人格：电台一部分是真录音，一部分是旧机构覆盖的协议指令，玩家需要分辨两种"声音"。
- **已引入可观测信号：** 3 类电台可疑信号设计（见 10-base-management-research.md），支持方案 B 或 C。
- **当前默认值：** 第一阶段使用方案 A（中立，但有盲区），为后续切换至 B 或 C 留门。

### Q-03 旧收容机构的核心罪行与最终真相

- **状态：** 🟡 可缓
- **来源模块：** 11-narrative-worldbuilding / 12-progression-difficulty-longterm-growth
- **影响范围：** 主叙事后期、真相闭合结局族群、旧机构真相完成度变量
- **问题描述：** 旧机构中期会揭示"曾经大量牺牲原形和人类来稳定副本"，但具体罪行性质、规模和玩家角色与此的关系需要细化。
- **可选方向：**
  - 核心罪行为"把人类当作收容材料"（暗合人体实验主题，强化恐怖感）
  - 核心罪行为"把异常塑形成工具并强制奴役"（暗合伦理冲突主题，强化原形关系意义）
  - 两者叠加：旧机构同时对人和异常进行了非伦理实验
- **当前默认值：** 暂不定案，第三阶段叙事深化时处理。第一阶段只需基础叙事骨架（见 11 模块第一阶段范围）。

### Q-04 结局族群进入阈值

- **状态：** 🟠 待决
- **来源模块：** 12-progression-difficulty-longterm-growth
- **影响范围：** 结局触发条件、长期倾向变量权重
- **问题描述：** 2 主轴矩阵确认了 4 大族群（对抗+揭露/对抗+继承/共生+揭露/共生+继承），但每个族群的具体进入阈值（变量积累到多少分数、倾向百分比如何计算）未定。
- **可选方案：**
  - 方案 A：固定阈值（如主轴 1 积分 ≥70% 才进入对应极）。清晰，但缺乏细腻感。
  - 方案 B：加权投票（各变量按权重投票，加总后投影到二维坐标轴）。更平滑，但实现复杂。
- **当前默认值：** 使用方案 A 作为第三阶段起点，第二阶段结束前细化阈值数字。

### Q-06 破败校园与旧机构最终真相的关系

- **状态：** 🟡 可缓
- **来源模块：** 12-progression-difficulty-longterm-growth / 11-narrative-worldbuilding
- **影响范围：** 真相闭合结局族群、首个副本叙事深度
- **问题描述：** 是否"破败校园"这个第一阶段副本本身就是旧机构罪行的直接现场？如果是，玩家第一局就能获取关键叙事锚点；如果否，破败校园只是异常副本之一，世界观揭示延迟到更后期。
- **可选方向：**
  - 校园是旧机构的第一批实验地点（高叙事密度，第一阶段有强世界观锚点）
  - 校园是独立的异常事件，与旧机构有间接关联（叙事压力较小，第一阶段聚焦玩法验证）
- **当前默认值：** 使用"间接关联"版本，第一阶段聚焦玩法验证，叙事悬念为后期保留。

### Q-13 像素分辨率全局基准

- **状态：** 🟠 待决
- **来源模块：** 00-tech-constraints §六
- **影响范围：** 所有美术资源与瓦片表
- **问题描述：** 像素美术瓦片基础尺寸需在垂直切片前定稿，避免后期资产返工。
- **可选方案：**
  - 方案 A 32×32：资源体量小，绘制成本低，但 2.5D 透视细节受限。
  - 方案 B 64×64：表现力更强，便于光影与异常细节，但每张资产工作量上升。
- **当前默认值：** 暂用 32×32 作为原型占位，定案后全局统一。

### Q-D02 结局结构方式

- **定案版本：** v0.6.0
- **决策：** 采用 2 主轴矩阵（与异常关系: 对抗↔共生 × 与旧机构关系: 揭露↔继承）确定 4 大结局族群，修饰位（原形关系/污染度/电台关系/真相完成度）在族群内部生成具体变体。6 个原始变量保留在数据层，不直接映射为独立结局。

- **定案版本：** v0.6.0

- **关联模块更新：** 08-origin-acquisition-growth v0.3.0

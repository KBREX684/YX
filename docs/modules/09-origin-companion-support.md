## 模块定位

原形携带与助战模块负责养成后的原形如何进入后续副本，并在探索、生存、击杀或收容中发挥作用。它让玩家的局外养成成果进入局内策略。

这个模块必须避免"带上最强原形即可通关"的单一最优解。每个原形能力都应有适用场景、代价和风险。

## 关联 Design Pillars

- **Pillar 1（看不见但可学习的怪物）：** 原形能力改变的是玩家理解和应对怪物规则的方式，而不是绕过它。拟人化原形提示（有误读概率）、恐怖化原形干扰规则（有污染代价）、工具化原形显示隐藏信息（有消耗代价），都服务于对怪物规则的学习与应用。
- **Pillar 2（任何变强都同时意味着变危险）：** 每种路线的副作用在高压环境中产生真实代价。致命伤抵消不是免费复活，而是路线特定代价（好感损耗/污染上升/工具受损）。原形越强，失控风险或副作用越重。

## 设计目标

- 让不同养成方向带来不同副本策略。
- 让原形能力参与探索、解谜、逃生、对抗和收容。
- 保持恐怖感，避免原形把玩家变成绝对安全。
- 让携带选择成为副本准备的重要决策。
- 让原形副作用在高压环境中产生戏剧性。

## 核心功能

### 携带规则

- 每次进入副本前选择可携带原形。
- 最多携带 2 个原形，第一阶段可以先实现 1 个携带位。
- 原形能力可能需要消耗素材、稳定度、冷却或污染值。
- 某些副本会限制特定类型原形。
- 原形可能因怪物规则受到压制、诱导或污染。
- 进入副本后无法临时切换原形。
- 原形能力包含主动技能和被动效果。

### 随行与外观形态

- 拟人型原形以实体随行，可以跟随玩家，默认表现为陪伴式移动，关键情境可出现牵手互动。
- 恐怖型原形化作纹身，附着在主角身上，通过皮肤纹路、发热、渗血或异动表现状态。
- 工具型原形保持道具外形，例如手电、地图、钥匙、耳饰等。

### 致命伤抵消与路线差异化代价

所有原形在副本中可以抵消一次致命伤害。触发后根据路线产生不同代价（与 08-origin-acquisition-growth.md 保持一致）：

| 路线 | 致命伤触发后代价 | 能力状态 |
|---|---|---|
| 拟人化 | 好感度 -50%；若好感度 < 20% 则拒绝主动提示 | 能力仍可用，但主动提示频率降低或拒绝触发 |
| 恐怖化 | 污染值 +25~+35；若超过阈值则进入不稳定状态 | 威慑能力可用，但副作用双倍，且失控风险提升 |
| 工具化 | 工具受损，关键功能部分失效 | 受损功能不可用，需要基地修复（1–2 次副本周期） |

触发致命抵消后，原形不进入完全休眠，而是"受损状态"——原形仍随行，但能力降级，体现代价而不是完全消失。

### 拟人化助战

可能能力：

- 提醒危险方向（主动技能，有冷却）。
- 帮助解读线索（被动，有误读概率）。
- 在玩家躲藏时稳定理智（被动，好感度 ≥ 50% 时生效）。
- 与副本中的人形异常交流（特定副本主题触发）。
- 触发专属剧情或隐藏房间（高好感度触发）。

可能副作用：

- 对某些线索产生主观误读（误读概率与好感度反相关）。
- 害怕特定怪物而短暂失联（与具体怪物类型相关）。
- 好感或信任不足时拒绝提示（好感度 < 20% 时禁用主动提示）。
- 过度依赖玩家，导致高压时干扰操作。
- 根据性格不同，拟人型原形可能在理智极低（< 20%）或好感度极低时触发失控判定。

### 恐怖化助战

可能能力：

- 短时间威慑低阶异常（打断追踪状态）。
- 打断怪物追击。
- 污染或破坏门锁、墙体、规则节点。
- 与怪物进行规则对抗（仪式辅助）。
- 为玩家争取仪式操作时间（拖延追猎切换到处置的窗口）。

可能副作用：

- 提高副本污染值（每次使用 +5 污染）。
- 吸引更危险异常（污染值高时触发）。
- 降低线索可靠性（污染值 > 60 时效果）。
- 失控后攻击环境或消耗资源。
- **理智极低（< 20%）时，恐怖型原形所有主动能力无效化**（见 04-horror-perception-pressure.md）。

### 工具化助战

可能能力：

- 活体手电：照出异常痕迹，能显示隐藏仪式位置。
- 会说谎的地图：显示隐藏路线，也会混入假路线。
- 封门钥匙：临时锁住门，也可能把玩家困住。
- 听墙耳饰：听见墙后声音，但会听到污染低语。

可能副作用：

- 需要喂养或充能（副本内消耗特殊素材）。
- 使用次数有限（每副本内固定次数）。
- 特定规则下反向误导玩家（感知欺骗类规则下可能出现假信息）。
- 过度使用会损害原形人格或稳定度。

## 关键流程

```
原形致命伤触发流程：

玩家受到致命伤害事件
  ↓
检查是否携带原形且原形未处于受损状态
  ├─ 是 → 消耗本局原形保护机会（一次）
  │    → 判断路线：
  │    ├─ 拟人化 → 好感度 -50%，触发拒绝判断
  │    ├─ 恐怖化 → 污染 +30，触发不稳定判断
  │    └─ 工具化 → 标记工具受损，关键功能不可用
  │    → 原形进入受损状态（能力降级但不消失）
  │    → 向 08 模块发送 fatal_trigger_event
  └─ 否 → 正常触发死亡流程
```

## 与其他模块关系

- **原形养成模块** ↔ 接收原形能力，同步致命伤代价
- **玩家控制模块** ↔ 共享随行、指令和能力触发
- **怪物规则模块** ↔ 进行克制、压制或冲突判断
- **地图模块** ↔ 互动隐藏路线、封门、破坏等空间功能
- **恐怖感知模块** ↔ 共同表现副作用（理智影响恐怖型）
- **目标结算模块** ← 记录原形使用造成的结果

## 第一阶段原型范围

第一阶段只实现 1 个原形的 3 种助战能力：

- 拟人化：危险提示，稳定或恢复一次理智。
- 恐怖化：短时间阻挡怪物接近。
- 工具化：活体手电，照出隐藏痕迹。

每种能力都需要一个副作用：

- 拟人化提示存在冷却和误读概率。
- 恐怖化增加污染。
- 工具化消耗特殊素材或稳定度。

致命伤代价三路线差异化完整实现（好感降/污染升/工具受损）。

## 验收标准（Acceptance Criteria）

- [ ] 玩家进副本前会根据目标选择不同原形路线（携带选择有意义）。
- [ ] 原形能力能改变副本策略，但不能直接解决所有问题。
- [ ] 副作用足够明确，玩家能感受到代价（不只是数字变化）。
- [ ] 理智极低（< 20%）时，恐怖型原形所有主动能力无效化可被观察验证。
- [ ] 好感度低于 20% 时，拟人型原形主动提示被禁用可被验证。
- [ ] 致命伤触发后，三种路线分别产生不同的可见代价。

## KPI 指标

| 指标名 | 目标值 | 测量方式 |
|---|---|---|
| 玩家携带原形进入副本的比率 | ≥60%（有原形时） | 数据统计 |
| 原形能力被使用的比率（有携带） | ≥70% | 数据统计 |
| 致命伤代价后玩家主动去基地修复的比率 | ≥50% | 数据统计 |

## 风险与依赖

- **依赖：** 08-origin-acquisition（原形路线、进度、稳定度）、04-horror-perception（理智值）、03-monster-anomaly-rules（怪物规则克制判断）。
- **风险：** R-01（体验曲线塌陷），原形能力必须有副作用而非纯增益，Pillar 2 规约；R-08（理智系统三方耦合），恐怖型无效化阈值与 04 模块共用同一阈值表。

## 后续扩展方向

- 多原形携带编队（第二阶段）。
- 原形之间的相性、嫉妒、共鸣和冲突。
- 原形专属副本互动（特定怪物类型的深度克制）。
- 原形被怪物污染或夺走。
- 玩家命令系统和原形自主行动。

## 已确认设计决策

- 原形助战包含主动技能和被动效果。
- 恐怖型原形化作纹身。
- 拟人型原形以实体跟随，可出现牵手陪伴表现。
- 工具型原形保持道具外形。
- 副本内无法临时切换原形。
- 理智低于 20% 时，恐怖型原形所有主动能力无效化。
- 好感度低于 20% 时，拟人型原形拒绝主动提示。
- 致命伤触发后代价差异化：拟人好感 -50%/恐怖污染 +30/工具受损，原形进入受损状态而非完全休眠。

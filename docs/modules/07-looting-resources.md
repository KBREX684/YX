版本：v0.3.2

## 模块定位

搜刮与资源模块负责副本内物资获取、消耗、携带和带回基地的经济循环。它决定玩家为什么要冒险深入危险区域，以及为什么不能只走最安全路线。

资源不应只是数值奖励，而应服务生存、解谜、收容和原形养成。

## 关联 Design Pillars

- **Pillar 1（看不见但可学习的怪物）：** 解谜物资（钥匙、仪式材料、录音带）和情报资源直接关联规则推理。玩家必须在危险中搜刮这些物品，才能理解和解决怪物。
- **Pillar 2（任何变强都同时意味着变危险）：** 出门预算制度和死亡惩罚机制确保资源决策始终有代价。玩家不能无风险地带走全部资源，也不能完全规避死亡损失。基地物资增加并不代表"更安全"，而代表"更多可选择的危险路线"。

## 设计目标

- 让资源点和危险区域形成明确取舍。
- 让玩家在生存消耗和长期养成之间做选择。
- 控制背包容量，避免玩家无脑收集全部物品。
- 让不同完成目标产出不同资源结构。
- 让出门准备成为重要策略决策，而不是简单"带满再走"。
- 让资源名称和用途符合异常收容题材。

## 核心功能

### 资源类型

生存物资：

- 电池（手电续航）。
- 绷带（伤势恢复）。
- 镇定剂（理智部分恢复）。
- 临时封门物（短时间阻挡怪物进入）。
- 噪音诱饵（吸引怪物注意，用于弱点链击杀）。

解谜物资：

- 钥匙。
- 符号碎片。
- 录音带。
- 仪式材料。
- 特殊房间通行物。

养成素材：

- 异常残渣。
- 记忆碎片。
- 器官样本。
- 污染结晶。
- 空壳原质。

情报资源：

- 档案页。
- 怪物记录。
- 基地研究资料。
- 旧收容日志。

### 背包与携带

- 背包采用格子制。
- 不同物资占据不同形状的格子，形成类似战术搜刮游戏的摆放取舍。
- 大型物品占用更多格子或需要双手携带。
- 仪式物可能无法放入普通背包。
- 高污染素材可能影响玩家理智（需要特殊容器）。
- 原形相关物品可能吸引怪物。

### 出门预算制度

解决 R-04（出门 Build 系统缺失）：玩家不能把整个仓库塞入背包出门，出门决策是准备阶段的核心策略之一。

背包格子分为两个区域：

**带入区（约 40% 格子，第一阶段固定比例）：**

- 玩家在基地准备时主动选择放入的物资。
- 受带入上限约束，不能超过该区域格子数。
- 死亡时，带入区物资 100% 损失（带进去的东西是被消耗的风险押注）。

**副本内拾取区（约 60% 格子）：**

- 仅用于副本内搜刮的物品。
- 出门时此区域为空，不允许预先填充。
- 死亡时，该区域物资约 60%–70% 返还至仓库（鼓励深入搜刮）。

**设计目的：**

- "带什么"变成策略决策（镇定剂 vs 噪音诱饵 vs 额外格子用于搜刮）。
- 深入搜刮有激励（拾取区死亡损失低），但带入资源风险高（全损）。
- 避免"满背包出门，直接完成副本"的无策略感。

### 风险收益

- 安全区域资源少。
- 危险区域资源更稀有。
- 高价值素材可能靠近怪物活动区。
- 某些资源需要制造声音或打开光源才能取得（感知风险）。
- 贪心搜刮会提高失败概率。

### 带回规则

- 玩家逃离、击杀或收容后可带回背包内资源。
- 死亡时：带入区（出门前选择的）100% 损失；副本内拾取的资源 60%–70% 返还。
- 特殊收容素材只有完成收容才能稳定带回（错误收容时可能损失这些素材）。
- 污染素材带回后需要基地处理，否则增加基地污染度。

### 错误收容资源惩罚

错误收容惩罚使用分级制度（见 06-objectives-settlement.md），不再是固定百分比，改为比例+绝对值上限组合：

- 轻度错误收容：min(仓库总量×5%, 3 单位) 损失。
- 中度错误收容：min(仓库总量×10%, 8 单位) 损失。
- 重度错误收容：min(仓库总量×15%, 15 单位) 损失。

使用绝对值上限原因：避免早期仓库空时惩罚失效（0×10%=0），以及后期仓库满时一次惩罚损失过重。

### 制作与特殊资源

- 副本内不允许临时制作资源，制作和改造主要发生在基地。
- 存在可交易或可牺牲的特殊资源，用于换取情报、仪式机会或异常协助。

## 关键流程

```
背包管理流程：

[副本前] 基地准备区：
  玩家从仓库选择物品 → 只能填入带入区（上限：40%格子）
  ↓
[副本中] 搜刮：
  玩家拾取物品 → 优先放入副本内拾取区（60%格子）
  → 拾取区满时，玩家可将带入区物品转入拾取区（腾出带入区格子用于更重要物品）
  ↓
[副本结束] 结算：
  正常完成 → 背包全部物资带回
  死亡 → 带入区物资清空；拾取区物资 65% 返还仓库
  错误收容 → 按分级惩罚表扣除仓库物资（不扣除带回的物资，扣的是已有仓库）
```

## 第一阶段原型范围

第一阶段需要实现：

- 3 种生存物资：电池、绷带、镇定剂。
- 3 种解谜物资：钥匙、录音带、仪式材料。
- 3 种养成素材：异常残渣、记忆碎片、污染结晶。
- 1 种情报资源：怪物档案页。
- 格子制背包，分带入区（约 40%）和拾取区（约 60%）。
- 死亡惩罚：带入区全损，拾取区 65% 返还。
- 副本结算时带回资源并更新仓库。

## 验收标准（Acceptance Criteria）

- [ ] 玩家在基地准备时会考虑"带什么"（带入区有上限，无法塞满）。
- [ ] 玩家会为了养成素材考虑是否深入危险区域（高价值素材在高危区域）。
- [ ] 生存物资能影响副本内即时决策（镇定剂在理智低时有意义）。
- [ ] 死亡时带入区全损，玩家感受到明显代价，但拾取区返还让深入探索有价值。
- [ ] 玩家无法无脑带走所有资源（格子物理限制）。
- [ ] 错误收容后仓库明确显示物资损失数量。

## KPI 指标

| 指标名 | 目标值 | 测量方式 |
|---|---|---|
| 带入区填满比率 | ≥70% 玩家在出门时填满带入区 | 数据统计 |
| 玩家主动进入危险区搜刮的比率 | ≥50% | 录像回放 |
| 死亡后带回资源（拾取区返还）被用于下次准备的比率 | ≥80% | 数据统计 |

## 风险与依赖

- **依赖：** 02-dungeon-generation（资源点）、06-objectives-settlement（结算方式和惩罚触发）、10-base-management（仓库状态）。
- **风险：** R-04（出门 Build 系统缺失），已通过带入区/拾取区双区格子制解决；R-05（死亡惩罚定义不明），已通过区分带入/拾取两类损失规则解决。

## 后续扩展方向

- 资源稀有度和品质分级。
- 素材污染处理（基地研究台处理污染素材）。
- 制作系统（基地内制作特殊道具）。
- 可丢弃、诱饵、临时封锁等战术物品。
- 原形偏好的素材和厌恶素材（增加养成方向策略性）。

## 已确认设计决策

- 背包采用格子制，分带入区（约 40%）和副本内拾取区（约 60%）。
- 出门预算制度：带入区格子有上限，出门前不能无限填充。
- 死亡惩罚区分：带入资源全损；副本内拾取资源约 65% 返还。
- 错误收容惩罚改为分级制度（轻/中/重），使用比例+绝对值上限。
- 不设置独立的失败型收容结算，只有错误收容。
- 资源不允许在副本中临时制作。
- 存在可交易或可牺牲的特殊资源。

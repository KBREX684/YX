## 用途

本目录用于存放游戏各系统模块的独立细化文档。总设定文档负责记录整体方向和模块关系，模块文档负责沉淀具体职责、功能范围、接口关系、原型范围和后续扩展。

后续每次修改模块文档时，需要同步更新该模块文档顶部版本号、最后更新时间和底部版本记录。如果修改影响整体结构，也要同步更新 `docs/game-concept.md`。

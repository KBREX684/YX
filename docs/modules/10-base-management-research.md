## 模块定位

基地经营与研究模块负责副本外的休整、资源管理、原形养成、怪物研究和下一次副本准备。它是玩家把副本成果转化为长期能力的空间。

基地应当相对安全，但不应完全安全。玩家把异常带回基地后，基地本身也会逐渐变得不稳定。

基地采用可自由移动场景，而不是纯菜单式据点。

## 关联 Design Pillars

- **Pillar 1（看不见但可学习的怪物）：** 研究台让玩家通过素材分析加深对怪物规则的理解，档案室提供跨副本的知识积累。这是基地作为"知识容器"而非"数值积累站"的体现。
- **Pillar 2（任何变强都同时意味着变危险）：** 基地污染度是"把副本风险带回家"的核心机制。基地入侵是最直接的代价体现：玩家选择更危险的路线，基地也变得更危险。NPC 不可信信号是另一层代价：信息越多，不确定性越高。

## 设计目标

- 给玩家提供副本后的喘息和整理空间。
- 承载原形养成、素材管理、研究解锁和剧情互动。
- 让基地升级影响后续副本准备。
- 保持异常存在感，避免基地变成纯菜单。
- 支持长期叙事和角色关系推进。

## 核心功能

### 原形收容室

- 存放原形（容量上限待定，见 Q-07）。
- 查看原形状态、稳定度、污染和进化方向。
- 进行投喂、安抚、压制、实验和进化。
- 触发原形异常事件。

### 研究台

- 分析副本素材。
- 解锁怪物弱点记录。
- 制作或改造特殊道具。
- 提升对某类异常的理解。
- 为后续副本提供准备加成。

### 仓库

- 管理生存物资、解谜物资、养成素材和情报资源。
- 支持整理、分解、封存和污染处理。
- 限制过量资源囤积，避免经济失控。
- 容量有上限（具体数字第二阶段确定）。

### 准备区

- 选择下一次副本携带道具（带入区物资）。
- 选择携带原形。
- 查看副本危险等级和已知情报。
- 制定目标倾向：逃离、击杀或收容。

### 档案室

- 查看怪物图鉴。
- 查看副本记录。
- 整理线索和世界观信息。
- 记录击杀、收容、失败和异常后果。

### 互动区

- 与拟人化原形互动。
- 触发角色剧情。
- 展示原形关系变化。
- 承载基地异常事件。

### 基地污染度（Contamination Level）

玩家没有电力、人员等传统基地管理资源。基地存在污染度，是基地"不完全安全"的主要机制。

**污染度因果表（缓解 R-09）：**

| 输入来源 | 增加量 | 说明 |
|---|---|---|
| 错误收容（轻度） | +3 | 仪式残缺 |
| 错误收容（中度） | +8 | 规则违反 |
| 错误收容（重度） | +15 | 完全错误顺序 |
| 恐怖型原形长期驻留（> 3 次副本） | +2/副本 | 稳定度越低，增加越多 |
| 副本带回高污染素材（未处理） | +1–+5/件 | 取决于素材污染等级 |
| 基地入侵失败（入侵者未被消除） | +10 | 异常在基地留下痕迹 |
| 拟人型原形失控事件 | +5 | 情感失控导致局部污染 |

| 输出影响 | 触发条件 | 具体效果 |
|---|---|---|
| 原形成长速度加快 | 污染度 30–60 | 路线偏向更快向恐怖化倾斜（反向：拟人化成长需要更多互动） |
| 原形性格偏移 | 污染度 > 50 | 拟人化原形的性格偏向不稳定或攻击性 |
| 基地异常事件频率上升 | 污染度 > 60 | 每次进入基地有概率触发小型异常事件 |
| 基地入侵概率提升 | 污染度 > 70 | 每次错误收容后触发入侵的概率提升 |
| 副本准备可靠性降低 | 污染度 > 80 | 准备区信息可能出现异常干扰（类似理智系统） |
| 基地全面崩坏 | 污染度 > 95 | 触发最终基地副本（见结局族群：基地崩坏） |

**污染度降低方式：**

- 研究台处理污染素材（-2~-5/次）。
- 成功完成收容并带回稳定原形（-3/次）。
- 使用特殊仪式物封印污染来源（-10，消耗稀有仪式材料）。

### 错误收容与基地入侵

**错误收容触发机制：** 见 06-objectives-settlement.md 分级惩罚表。错误收容同时会按概率触发基地入侵事件。

**基地入侵 spec（缓解 R-06 基地入侵设计缺失）：**

基地入侵是独立玩法单元，与常规副本共用引擎但有独有规则：

| 项目 | 规则 |
|---|---|
| 携带限制 | 不可携带原形进入入侵应对 |
| 手电 | 手电时间有限制（比常规副本更短） |
| 仓库风险 | 入侵者在仓库区域活动时，特定物资有破坏概率 |
| 入侵怪物规则 | 入侵者遵循与原副本不同的规则（简化版或变体规则） |
| 目标 | 躲藏求生（等待入侵者离开）或使用特殊道具击杀 |
| 时间限制 | 比常规副本短（约 5–8 分钟） |
| 失败后果 | 入侵失败 → 额外仓库损失 + 污染度 +10 |
| 成功奖励 | 入侵成功击杀 → 少量素材 + 污染度 -5 |

**基地入侵是将副本失败风险带回局外循环的核心机制（Pillar 2 体现）。**

### NPC 与电台

- NPC 存在，但仅限于引导和电台沟通。
- NPC 不是实体，不在基地中直接出现。

**电台不可信信号（3 类，支持叙事层不确定性）：**

| 信号类型 | 表现方式 | 玩家可验证方式 |
|---|---|---|
| 与档案矛盾 | 电台陈述内容与玩家找到的档案文本存在直接矛盾 | 对比档案室记录和电台对话记录 |
| 预测失败 | 电台预测"该区域今天没有异常"但怪物依然出现 | 录像回放或多次观察对比 |
| 关键时刻沉默 | 玩家面临关键决策（如启动收容仪式）时电台突然断联 | 模式观察（是否重复在关键时刻断联） |

这三类信号让电台可信度成为玩家自主探索的叙事层（见 Q-02），不强制指定答案。

## 关键流程

```
副本结算后基地更新流程：

接收副本结算数据（资源/原形/错误收容/入侵判定）
  ↓
更新仓库（带回资源 + 错误收容扣除）
  ↓
更新原形状态（原形入收容室 OR 更新受损状态）
  ↓
[错误收容] → 更新基地污染度（分级增加）
           → 判断入侵概率 → [触发] 启动基地入侵副本
  ↓
更新档案室
  ↓
检查基地污染度阈值 → 触发对应输出影响（异常事件/原形偏移/崩坏警告）
  ↓
玩家在基地进行：原形养成 / 研究台分析 / 准备区装备选择
```

## 与其他模块关系

## 第一阶段原型范围

第一阶段基地是可自由移动的小场景，需要包含：

- 原形收容室（容量 3）。
- 仓库（含污染素材标识）。
- 准备区（携带道具和原形选择）。
- 档案入口。
- 基地污染度显示（视觉提示，不一定显示数值）。

最小功能：

- 查看带回资源。
- 选择原形养成方向（投喂）。
- 查看怪物档案进度。
- 选择下一次进入副本时携带的原形能力。
- 记录基地污染度（随错误收容累积）。
- 基地入侵流程可触发（简化版：只有躲藏选项，无击杀选项）。

## 验收标准（Acceptance Criteria）

- [ ] 玩家完成副本后能在基地看到成果变化（仓库、原形状态、档案）。
- [ ] 玩家能用素材推动原形成长。
- [ ] 玩家能从基地准备影响下一次副本。
- [ ] 基地不是单纯结算页，而有明确"异常被带回来了"的感觉（污染度视觉提示）。
- [ ] 错误收容后基地污染度可见增加。
- [ ] 基地入侵事件可以被触发（即使第一阶段只有躲藏选项）。
- [ ] 电台 NPC 的三类不可信信号至少有一类在第一阶段出现过。

## KPI 指标

| 指标名 | 目标值 | 测量方式 |
|---|---|---|
| 玩家在基地停留时间 | 3–7 分钟/次（不能太短=无深度，不能太长=无聊） | 录像计时 |
| 玩家主动查看怪物档案的比率 | ≥60% | 数据统计 |
| 玩家注意到污染度变化的比率 | ≥70%（有错误收容时） | 测试后访谈 |

## 已确认设计决策

- 基地是可自由移动场景。
- 错误收容会导致怪物入侵，基地会化身特殊副本（简化版副本）。
- 入侵时玩家选择躲藏（第一阶段）或使用特殊道具击杀（第二阶段）。
- 玩家没有电力、人员等传统基地管理资源，但有基地污染度。
- 污染度因果表已定义（输入源、增加量、输出影响）。
- NPC 存在，但仅限引导和电台沟通，不是实体。
- 电台不可信信号共 3 类（与档案矛盾/预测失败/关键沉默）。
